# lieroy
Lie algebra library for both Python and C++

## Getting started

### Dependencies

Lieroy has Eigen3 as a dependency.

```
sudo apt install libeigen3-dev
```

### Installation

Build and install the library using cmake.

```
git clone git@github.com:davidlandry93/lieroy
cd lieroy
mkdir build
cd build
cmake ..
make
sudo make install
```
